## 3. Debug 记录（我发现这次作业相比上次ai生成的东西多了很多错误，我就边做边记录把他们都整理下来了，助教老师可以只选择前两个看要不然这工作量bushi）

这次作业在真正完成代码之前，经历了多次调试。调试过程中我发现，题目描述、实际数据文件、运行环境和绘图环境之间并不是完全一致的，所以不能只照着题面机械写代码，必须先运行、先看结果、再修正。下面记录几次最关键的 Debug 过程。

### Debug 1：按题面用制表符读取后，数据只剩1列

一开始我严格按照题目要求写成：

```python
df = pd.read_csv('ICData.csv', sep='\t')

但运行后发现输出结果不对：

数据集列数变成了 1
表头不是正常的 10 个字段，而是整串
交易类型,交易时间,交易卡号,刷卡类型,...
后面一访问 df['交易时间'] 就报错：KeyError: '交易时间'

这说明虽然题面写的是“制表符分隔”，但我手里的这份真实文件并不是按 \t 分隔的。
为了确认这个判断，我先打印了前5行和列数，最后发现文件实际上是逗号分隔。因此我把读取方式改成：

df = pd.read_csv('ICData.csv', encoding='utf-8-sig')

修改以后，数据恢复成正常的 10 列，问题解决。
这一步让我意识到，做数据分析时不能只相信说明文档，还要先验证真实数据格式。

Debug 2：时间格式和题面不一致，最开始解析格式写错

题目中给出的时间格式写成 YYYY/M/D HH:MM，所以我一开始写的是：

df['交易时间'] = pd.to_datetime(df['交易时间'], format='%Y/%m/%d %H:%M')

但实际查看数据后发现，时间字段像这样：

2018/4/1 11:45:12

也就是说，这份数据里的时间是带秒的。
如果还按不带秒的格式去解析，后面就会出现时间格式不匹配的问题。

所以我把代码改成：

df['交易时间'] = pd.to_datetime(df['交易时间'], format='%Y/%m/%d %H:%M:%S')

这样改完之后，时间列就能正确转成 datetime 类型，后面提取 hour 字段也能正常进行。

Debug 6：任务2和任务6都有中文字体 warning

在画任务2柱状图和任务6热力图时，一开始都出现了中文字体 warning，例如提示某些中文字符在默认字体 DejaVu Sans 中缺失。虽然这些 warning 不会直接让程序报错，但会影响图表显示，也会让运行结果显得不够干净。

我先对 matplotlib 做了中文字体设置，例如指定 Windows 系统里的中文字体文件，并关闭负号乱码问题。
但后来发现，任务6的 heatmap 还有一个更隐蔽的问题：seaborn.heatmap() 会在内部先根据 DataFrame 的行索引去绘图。如果这个索引一开始就是中文，seaborn 在内部 draw 的时候还是可能先用默认字体，于是中文 warning 依旧会冒出来。

最后我的处理方法是：

先用英文索引构造 heatmap_data
等图画出来以后，再手动把 y 轴刻度改成中文
然后给这些标签单独设置 fontproperties=myfont

这样做之后，任务6的字体 warning 就消失了，图像也能正常显示中文。

4. 人工代码审查（逐行中文注释）

下面贴出任务4中 PHF 计算的核心代码，并加上我自己理解后的逐行中文注释。

# 这里继续沿用前面筛好的上车刷卡数据
# 这样口径一致，都是刷卡类型=0
hourly_counts = df_bus.groupby('hour').size().reindex(range(24), fill_value=0)

# 找出全天刷卡量最大的那个小时
peak_hour = hourly_counts.idxmax()
peak_hour_count = hourly_counts.max()

# 只保留这个高峰小时里的记录
peak_df = df_bus[df_bus['hour'] == peak_hour].copy()

# 把交易时间往下归到所在的5分钟时间段
# 比如 09:17:36 会被归到 09:15:00
peak_df['time_5min'] = peak_df['交易时间'].dt.floor('5min')

# 把交易时间往下归到所在的15分钟时间段
peak_df['time_15min'] = peak_df['交易时间'].dt.floor('15min')

# 统计高峰小时里每个5分钟窗口的刷卡量
count_5min = peak_df.groupby('time_5min').size().sort_index()

# 找到刷卡量最大的那个5分钟窗口
max_5min_time = count_5min.idxmax()
max_5min_count = count_5min.max()

# 按题目公式计算 PHF5
# PHF5 = 高峰小时刷卡量 ÷ (12 × 高峰小时内最大5分钟刷卡量)
phf5 = peak_hour_count / (12 * max_5min_count)

# 统计高峰小时里每个15分钟窗口的刷卡量
count_15min = peak_df.groupby('time_15min').size().sort_index()

# 找到刷卡量最大的那个15分钟窗口
max_15min_time = count_15min.idxmax()
max_15min_count = count_15min.max()

# 按题目公式计算 PHF15
# PHF15 = 高峰小时刷卡量 ÷ (4 × 高峰小时内最大15分钟刷卡量)
phf15 = peak_hour_count / (4 * max_15min_count)

# 下面把时间格式整理成题目要求的展示样式
peak_hour_start = f"{peak_hour:02d}:00"
peak_hour_end = f"{(peak_hour + 1) % 24:02d}:00"

max_5min_start = max_5min_time.strftime('%H:%M')
max_5min_end = (max_5min_time + pd.Timedelta(minutes=5)).strftime('%H:%M')

max_15min_start = max_15min_time.strftime('%H:%M')
max_15min_end = (max_15min_time + pd.Timedelta(minutes=15)).strftime('%H:%M')

print("\n任务4 高峰小时系数计算结果：")
print(f"高峰小时：{peak_hour_start} ~ {peak_hour_end}，刷卡量：{peak_hour_count} 次")
print(f"最大5分钟刷卡量（{max_5min_start}~{max_5min_end}）：{max_5min_count} 次")
print(f"PHF5  = {peak_hour_count} / (12 × {max_5min_count}) = {phf5:.4f}")
print(f"最大15分钟刷卡量（{max_15min_start}~{max_15min_end}）：{max_15min_count} 次")
print(f"PHF15 = {peak_hour_count} / ( 4 × {max_15min_count}) = {phf15:.4f}")
